# PDF Anchor Detection Library

This library provides functionality to find text anchors in PDF documents and calculate positions for inserting signatures or other elements.

## Features

- Find text anchors in PDF documents
- Calculate signature positions based on anchors
- Support for multiple anchor detection
- Configurable matching thresholds and positioning offsets
- Regular expression support for anchor detection
